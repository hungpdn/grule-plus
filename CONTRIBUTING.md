# Contributing to grule-plus

Thank you for your interest in contributing to grule-plus! We welcome contributions from the community.

## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Make your changes** following the coding standards.
3. **Add tests** for new features or bug fixes.
4. **Run tests and linting**: `make test` and `make lint`.
5. **Update documentation** if necessary.
6. **Submit a pull request** with a clear description of your changes.

## Coding Standards

- Follow Go conventions (use `gofmt` and `goimports`).
- Write clear, concise commit messages.
- Add comments for complex logic.
- Ensure all tests pass.

## Reporting Issues

Use the issue templates in `.github/ISSUE_TEMPLATE/` to report bugs or request features.
